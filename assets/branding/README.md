# Brand Assets

These files are the public-safe, first-party branding assets for `repo-harness`.

- `docs/images/repo-harness-banner.svg`: English README and social-share banner. The SVG file is the editable source.
- `docs/images/repo-harness-banner-cn.svg`: Chinese README and social-share banner. The SVG file is the editable source.
- `docs/images/repo-harness-controller-flow.svg`: GitHub-renderable controller architecture diagram export.
- `repo-harness-controller-flow.mmd`: editable Mermaid source for the controller architecture diagram.

Color palette:

- `#081120` / `#102546`: primary deep navy background
- `#3DD9C5`: controller/flow accent
- `#FFB347`: repository/evidence accent
- `#406AA4`: connector arrow accent
- `#311C53`: agent execution accent

Usage notes:

- Prefer the SVG files in `docs/images/` when embedding images in README files.
- Prefer the Mermaid file when changing the architecture structure first, then update the exported SVG to match.
- Do not introduce third-party logos, trademarks, or stock illustrations into these assets.
